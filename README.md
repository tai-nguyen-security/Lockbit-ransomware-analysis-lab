# LockBit Ransomware v4.0 Malware Analysis Lab

Role: Reverse Engineer (Team project)

This project presents static and dynamic analysis of a LockBit ransomware sample in a controlled virtual lab environment.

## Key Activities

- Performed static analysis on PE structure and entropy
- Conducted dynamic behavior monitoring in isolated VM
- Identified Indicators of Compromise (IOC)
- Analyzed persistence and anti-debugging techniques
- Built sandbox environment using VMware

## Tools Used

IDA Pro
Ghidra
x64dbg
Detect It Easy (DIE)
PE Explorer
VMware Workstation

## Environment

Windows Server VM
Kali Linux
Isolated sandbox lab
